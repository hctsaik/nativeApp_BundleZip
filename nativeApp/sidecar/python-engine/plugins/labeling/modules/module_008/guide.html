什麼是 Anchor 幀？

Anchor 幀是你親手標注 bounding box 的那一幀影片畫面。系統會以這一幀為基準，自動將 bbox 往前後傳播到你設定的時間範圍。

💡 如何選好 Anchor 幀？

• 選動物清晰可見、沒有模糊的幀

• 選動物位於畫面中央、沒有被遮擋的幀

• 避免選動物快速移動、畫面模糊的瞬間

③ 在 X-AnyLabeling 畫 Bounding Box

操作步驟

1. 點擊 「🖊 在 X-AnyLabeling 畫框」 按鈕 → X-AnyLabeling 視窗開啟

2. 在圖片上的目標部位拖拉畫出矩形框

3. 為每個框選擇對應的標注類別（如：眼睛、鼻子）

4. X-AnyLabeling 有自動存檔功能，儲存後回到這個頁面等待更新（約 2 秒）

✅ 已偵測到框的樣子

頁面自動刷新後，你會看到 「✅ 已偵測到 N 個標注框，點選上方▶執行開始追蹤傳播」 的綠色提示。

⚠️ 畫框注意事項

• 每個框要緊貼物件邊緣，不要留太多空白

• 可以畫多個框（例如左眼、右眼各一個框）

• 儲存後回到頁面，如果沒有更新請稍等 2-3 秒讓 autorefresh 讀取

④⑤ 設定時間範圍並啟動追蹤

往前幾秒 / 往後幾秒

設定從 Anchor 幀出發，往前/後各追蹤幾秒的影片片段。

例如：往前 1 秒 + 往後 1 秒 = 總共 2 秒的追蹤範圍。

30fps 的影片中，2 秒 ≈ 60 幀。

設定完成後，確認頁面下方出現綠色提示「✅ 已偵測到標注框」，再點選上方 ▶ 執行。

系統啟動後會顯示「執行完成，請切換至 Output 頁籤查看結果」，請切換至 Output 頁籤。

Output 頁面說明

⏳

追蹤進度

顯示「追蹤中… X/Y 幀」和進度條，完成前每 1.5 秒自動更新。

🖼

時間軸縮圖

所有追蹤幀以縮圖排列，附信心分數徽章。點擊任一縮圖可進入詳細檢視。

🔍

幀詳細檢視

顯示原圖 + 追蹤結果疊合圖，以及各 bbox 的座標與信心值。

✅

手動校正

若某幀的追蹤結果不準確，可調整 bbox 座標後點「確認校正」儲存。

🔄

從此幀重新傳播

校正後點此按鈕，系統會從你校正的這幀重新往前後追蹤，修正後續偏移。

📤

匯出（X-AnyLabeling JSON）

點擊匯出按鈕，將所有幀的 bbox 打包成 X-AnyLabeling 格式，存至 exports/xanylabeling/。

信心分數說明

每個追蹤幀都有一個 0～1 的信心分數，表示追蹤結果的可靠程度。

🟢 ≥ 0.7  高信心—追蹤準確
🟡 0.4–0.7  中信心—建議手動確認
🔴 < 0.4  低信心—建議手動校正

⚠️ 低信心幀怎麼處理？

在 Output 頁面的時間軸上，低信心幀（🔴）會特別顯眼。點入該幀，手動調整 bbox 後，點「🔄 從此幀重新傳播」，系統會以你校正的結果重新追蹤後續幀。

常見問題

Q：點了「執行」，但 Output 頁面顯示「請先在 Input 頁面選擇影片並啟動追蹤」？
A：請確認 Input 頁面下方有出現綠色的「✅ 已偵測到 N 個標注框」提示，再點「▶ 執行」。若尚未在 X-AnyLabeling 畫框，系統不會啟動追蹤。
Q：X-AnyLabeling 畫完框存檔了，但頁面還是顯示「尚未偵測到標注」？
A：頁面每 2 秒自動刷新一次。請稍等，或手動重新整理 Input 頁面。若等待 10 秒以上仍無反應，請確認 X-AnyLabeling 已正確開啟並存檔。
Q：追蹤速度很慢？
A：若機器未安裝 PyTorch（torch），系統會退回到「僅 Optical Flow」模式，速度較快但精度較低。Output 頁面會顯示 ⚠️ 提示。
Q：匯出的 JSON 檔案在哪裡？
A：匯出後的 JSON 存放在：{工作區根目錄}/{session_id}/exports/xanylabeling/。Output 頁面匯出成功後會顯示完整路徑。
Q：如何修改「眼睛、鼻子、嘴巴」這些預設標注類別？
A：標注類別由 Module 008 獨立設定（config/module_008.json）管理。請聯絡管理員調整，或直接修改設定檔中的 annotation_labels 欄位。
