① 設定資料夾路徑

📌 必填項目

影像資料夾路徑：包含動物圖片的資料夾（預設 testData/animal/）

資料庫路徑：存放標記記錄的 SQLite 檔案（預設 testData/animal/animals.db）

⚠️ 最常犯的錯誤

路徑打錯，或資料庫尚未建立。如果按下「執行」後看到「資料庫不存在」，請確認 animals.db 的路徑是否正確。

② 執行並載入影像清單

填好路徑後，點選上方 ▶ 執行 按鈕。系統會從資料庫讀取影像清單，顯示在右側表格。

✅ 成功的樣子

右側顯示「共 N 筆 | 已分類 M 筆 | 已標注 K 筆」統計列，以及可點擊的影像清單表格。

③ 從清單選擇影像

在表格中點擊任意一列，右側就會出現該影像的預覽。

可用上方的 類別篩選（全部 / 貓 / 狗 / 大象）快速縮小範圍，找出尚未標記的影像。

④ 分類標記與 Bounding Box 標注

4a — 分類標記

在影像預覽右側選擇正確類別，然後點選 Submit。系統會自動跳到下一張尚未分類的影像。

✅ 正確
框剛好包圍動物
沒有多餘空白
❌ 避免
框比動物大很多
影響追蹤精度
❌ 避免
框裁切到動物
部分身體不在框內

4b — Bounding Box 標注

點擊 🏷 標注 按鈕進入標注畫布，在動物身上的目標部位拖拉畫框。

標注類別顏色對應

貓
狗
大象
unknown

💡 標注小技巧

• 同一張影像可以畫多個框（例如：眼睛、鼻子各一個框）

• 畫完所有框後，務必點 💾 儲存標注，否則重新整理頁面後會消失

• 畫錯了？點 🗑 清除全部 重來

⑤ 儲存後自動跳至下一張

點選 Submit 或 💾 儲存標注 後，系統會自動跳至下一張尚未完成的影像。

統計列（共 N 筆 | 已分類 M 筆 | 已標注 K 筆）會即時更新，讓你知道進度。

修改標注類別

預設標注類別為「眼睛、鼻子、嘴巴」。若需要新增或修改，請聯絡管理員調整 Module 006 設定（位於 config/module_006.json）。

常見問題

Q：點擊「執行」後畫面空白，什麼都沒有？
A：請確認「資料庫路徑」設定正確，且 animals.db 確實存在。也可嘗試點選「進階：路徑設定」手動輸入完整路徑。
Q：儲存標注後，頁面沒有自動跳到下一張？
A：如果所有影像都已標注完畢，系統就不會再跳了。查看統計列確認「已標注」數量是否等於總筆數。
Q：標注框畫了沒有反應？
A：請確認你已點擊 🏷 標注 進入標注模式（按鈕會變成「🏷 標注中」）。在預覽模式下無法畫框。
Q：匯出的標注檔在哪裡？
A：每張影像的標注以 {檔名}_annotations.json 的格式，儲存在與影像相同的資料夾中。
